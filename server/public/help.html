Quick Start: Missing Textures?

Why is my model white/grey?

If you uploaded an .obj file and the result has no color, you likely forgot the texture files!

The Rule: If you upload an OBJ file, you MUST also select the texture images (JPG/PNG) and the material file (MTL) at the same time.

How to fix it: In the upload window, select ALL files (model.obj, model.mtl, texture.jpg) and drag them in together. Or, use a .glb file which has textures built-in.

Which Mode Should I Use?

Standard Decimation

Cost: 1 Credit

Best for: Reducing file size while keeping the exact same geometry structure. Good for CAD models or models that are already clean.

How it works: It simply removes polygons. It does not create new textures.

Premium Remesh (Bake)

Cost: 5 Credits

Best for: 3D Scans, Photogrammetry, and Organic shapes.

How it works: It builds a brand new "clean" mesh (Retopology) and "bakes" the details from the high-quality original onto the new one using Normal Maps. This is the professional workflow for games and AR.

Supported File Formats

  • GLB / GLTF: The best format. Textures are embedded.
  • OBJ: Good, but requires you to upload .mtl and .jpg/.png textures alongside it.
  • FBX: Supported, but textures can sometimes be tricky if they aren't embedded in the FBX.
  • ZIP: You can zip your OBJ+MTL+Textures into a single file and upload that.

Pricing & Credits

We use a simple credit system. You purchase a pack of credits, and they are deducted when you run a job.

  • Failed Jobs: If the server fails to process your file, your credits are automatically refunded.
  • Re-runs: Optimizing the exact same file again within 24 hours (e.g., to try a different ratio) is FREE.
