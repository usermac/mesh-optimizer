Understanding Polygon Budgets

  • Scene Budget Total triangles for everything visible. Includes models, environment, and UI elements.
  • Per-Model Budget Maximum triangles for a single model. Divide scene budget by visible model count.
  • Texture Memory GPU memory for textures. Mobile devices have 1-2GB limits; exceed this and the app crashes.
  • Draw Calls Commands sent to GPU. Each material/mesh combo = 1 draw call. Keep under 100 for mobile.
  • File Size Download size matters for web. Users abandon loads over 5 seconds. Target under 5MB for mobile.
