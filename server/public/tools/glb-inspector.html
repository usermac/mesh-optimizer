What This Tool Shows You

  • Total Polygons Triangle count across all meshes. Critical for performance on web and mobile.
  • Mesh Count Number of separate mesh objects. More meshes = more draw calls.
  • Materials Each material adds rendering overhead. Consolidate when possible.
  • Textures Texture count and memory. Large textures are often the biggest bottleneck.
  • Animations Animation clips, duration, and bone count for skeletal animations.
  • File Size Compressed GLB size. For web delivery, aim for under 5MB.
