Quick Start: Missing Textures?

Why is my model white/grey?

If you uploaded an .obj file and the result has no color, you likely forgot the texture files!

The Rule: OBJ files need their associated material file (.mtl) and texture images (.jpg/.png) to display colors correctly.

How to fix it:

Option 1 (Recommended): Create a .zip file containing your model.obj, model.mtl, and all texture images. Drag the ZIP onto the upload area.

Option 2: Drag just the .obj file, then use the "Material (.mtl)" and "Textures" fields that appear below to add the additional files separately.

Option 3: Use a .glb file instead — it has textures built-in, so you only need one file.

Which Mode Should I Use?

Standard Decimation

Cost: 1 Credit

Best for: Reducing file size while keeping the exact same geometry structure. Good for CAD models or models that are already clean.

How it works: It simply removes polygons. It does not create new textures.

Premium Remesh (Bake)

Cost: 5 Credits

Best for: 3D Scans, Photogrammetry, and Organic shapes.

How it works: It builds a brand new "clean" mesh (Retopology) and "bakes" the details from the high-quality original onto the new one using Normal Maps. This is the professional workflow for games and AR.

Supported File Formats

  • GLB / GLTF: The best format. Textures are embedded. Just drag and drop.
  • ZIP (Recommended for OBJ): Zip your .obj + .mtl + textures into one file. Easiest way to keep everything together.
  • OBJ: Supported. After uploading, use the Material and Textures fields to add .mtl and .jpg/.png files separately.
  • FBX: Supported, but textures may not transfer if they aren't embedded in the FBX.

Quick Start Example

# 1. Submit a job
curl -X POST 'https://webdeliveryengine.com/optimize' \
     -H 'Authorization: Bearer YOUR_API_KEY' \
     -F 'file=@model.zip' \
     -F 'mode=decimate' \
     -F 'ratio=0.5'

# 2. Poll for completion (replace JOB_ID)
curl 'https://webdeliveryengine.com/job/JOB_ID'

# 3. Download results when complete
curl -O 'https://webdeliveryengine.com/download/JOB_ID/model_opt.glb'
curl -O 'https://webdeliveryengine.com/download/JOB_ID/model_opt.usdz'
